# Contributing to Stock Trader

Thank you for your interest in contributing to Stock Trader! This document provides guidelines and instructions for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/stock-trader.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test your changes locally
6. Commit: `git commit -m "Add your feature"`
7. Push: `git push origin feature/your-feature-name`
8. Open a Pull Request

## Development Setup

Follow the installation instructions in the main README.md file.

## Code Style

- **Python**: Follow PEP 8 style guide
- **TypeScript/React**: Use consistent formatting, prefer functional components
- **Commits**: Write clear, descriptive commit messages

## Testing

Before submitting a PR, please:
- Test all functionality locally
- Ensure the backend and frontend both run without errors
- Check that your changes don't break existing features

## Pull Request Process

1. Update the README.md if needed
2. Ensure your code follows the project's style
3. Test thoroughly
4. Submit the PR with a clear description
